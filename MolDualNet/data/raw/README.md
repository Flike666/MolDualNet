# Raw Datasets

Place the following MoleculeNet CSV files in this directory:

- `ESOL.csv` — aqueous solubility (log S)
- `FreeSolv.csv` — hydration free energy (kcal/mol)
- `Lipophilicity.csv` — octanol-water partition coefficient (log D)
- `BACE.csv` — beta-secretase 1 inhibitory potency (pIC50)

Download from: https://moleculenet.org
